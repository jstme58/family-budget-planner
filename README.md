# 🏠 Family Budget Planner

A beautifully designed, fully interactive family budget planner 
built with pure HTML, CSS, and vanilla JavaScript. No frameworks, 
no dependencies, no database — just open and use.

## ✨ Features

- 💵 Household income tracker (primary, secondary, other)
- 🏠 Expense sections for housing, food, transport, family & financial goals
- 📊 Live spending breakdown with animated progress bars
- ⚡ Personalized Quick Wins tips based on your actual numbers
- 🌍 5 languages & 8 currencies supported
- 📱 Fully responsive on mobile, tablet, and desktop
- 🔒 100% private — no data stored, everything runs in the browser
